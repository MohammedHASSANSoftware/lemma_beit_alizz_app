# Lemma Beit Alizz App
تطبيق لمة بيت العز هو تطبيق متخصص لعرض قوائم الطعام وطلبات التوصيل الخاصة بمطعم **بيت العز**، مع واجهة سهلة الاستخدام وتجربة سلسة للعميل.

## المميزات
- عرض قائمة الطعام بشكل منظم.
- خيارات تخصيص الطلب.
- سلة مشتريات سهلة التحكم.
- تتبع حالة الطلب.
- دعم متعدد اللغات.
- واجهة بسيطة وأنيقة.

## هدف التطبيق
يهدف التطبيق لتقديم تجربة طلب سهلة وسريعة لعملاء مطعم بيت العز، والوصول السريع لخدمات المطعم وتحديثاته.

## التقنيات المستخدمة
- Flutter
- Dart
- Firebase (في حال تم الربط)
- GitHub للتطوير وإدارة الإصدارات
